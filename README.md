# NinjaRunV4

## Overview
NinjaRunV4 is a 2D sidescroller project developed using Unreal Engine. This game features fast-paced action and various challenges that will test the player's skills as they navigate through vibrant levels filled with obstacles.

## Features
- **2D Sidescrolling Action**: Engage in dynamic gameplay that keeps players on their toes.
- **Multiple Levels**: Explore different environments and face unique challenges in each level.
- **Character Customization**: Customize your ninja's appearance and abilities.
- **Leaderboards**: Compete with friends for the highest scores.

## Getting Started
To get started with the NinjaRunV4 project, follow these instructions:
1. Clone the repository:
   ```bash
   git clone https://github.com/Laksidecloudz/NinjaRunV4.git
   ```
2. Open the project in Unreal Engine.
3. Build and run the project.
